# Backend of Digikala

--- 

## Run Project :

- `docker compose up `

---
## Fetures : 

- ### Using JWT authentication 
- ### Send OTP 
- ### Swagger for API
- ### Dockerized
- ### Testing 
- ### Using `Redis` as Message Broker and Result Backend
- ### Using `Celery` for async tasks
- ### Limit request rate with `Throttling`
